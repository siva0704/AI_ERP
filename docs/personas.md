# User Personas & Features - AI SMS

## 1. Admin
**Role:** School Principal, IT Administrator, or System Manager.
**Goal:** Oversee entire school operations, manage users, and configure system settings.
**Key Features:**
- **Dashboard:** High-level overview of attendance, fees, and school performance.
- **User Management:** Create/Edit/Delete Student, Teacher, and Parent accounts.
- **Timetable Management:** Create and assign class schedules.
- **Fee Management:** Configure fee structures, view payments, and send reminders.
- **Reports:** Generate system-wide reports (Attendance, Performance, Financial).

## 2. Teacher
**Role:** Class Teacher or Subject Teacher.
**Goal:** Manage daily class activities, track student progress, and communicate with parents.
**Key Features:**
- **Attendance:** Mark and view student attendance.
- **Timetable:** View personal class schedule.
- **Marks/Grades:** Enter exam marks and generate report cards.
- **Assignments:** Upload assignments and track submissions.
- **Communication:** Send messages to parents/students.

## 3. Student
**Role:** Enrolled Pupil.
**Goal:** Stay updated with school activities, classes, and performance.
**Key Features:**
- **Dashboard:** View upcoming classes, assignments, and notices.
- **Timetable:** Personal class schedule.
- **Attendance:** View own attendance records.
- **Results:** View exam results and report cards.
- **Profile:** Manage personal details.

## 4. Parent
**Role:** Guardian of the Student.
**Goal:** Monitor child's progress, attendance, and pay fees.
**Key Features:**
- **Dashboard:** Overview of child's activities.
- **Attendance:** View child's attendance history.
- **Fee Payment:** View dues and pay fees online.
- **Results:** Access child's report cards.
- **Communication:** Chat with teachers or school admin.
